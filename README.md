# 📊 Sales Dashboard - Power BI

## Project Overview

This project is an interactive Sales Dashboard built using Microsoft Power BI.

The dashboard helps businesses analyze sales performance, products, cities, employees, and yearly trends.

## Features

✔ Total Sales KPI

✔ Total Orders KPI

✔ Most Ordered Product

✔ Profit Analysis

✔ Sales by Product Line

✔ Sales by Product Name

✔ Quantity Ordered by Year

✔ Product Performance

✔ Interactive Map

✔ Dynamic Filters

---

## Filters Available

- Order Year
- City
- Employee
- Product Line

## Technologies Used

- Microsoft Power BI
- Excel
- DAX
- Power Query

## Dashboard Insights

The dashboard allows users to:

- Identify the best-selling products
- Monitor yearly sales
- Compare product categories
- Track employee performance
- Analyze sales geographically
- Filter data dynamically
